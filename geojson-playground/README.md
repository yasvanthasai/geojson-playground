# GeoJSON Playground

Simple React app to visualize GeoJSON data on Google Maps.

## Setup

1. Clone and install:
```bash
npm install
```

2. Add your Google Maps API key:
```bash
cp .env.example .env
```

3. Run:
```bash
npm run dev
```

## What it does

- 4 tabs showing different GeoJSON types
- Click map features to see details
- Basic points, lines, and polygons
